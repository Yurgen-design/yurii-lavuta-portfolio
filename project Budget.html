Budget Planner

Role

UI/UX Designer

Tools

Figma, UX Research, Prototyping, UX Testing

Budget Planner is a minimalist and intuitive web application for personal finance management. The platform helps users track income, detail expenses, and plan savings through visual analytics and a user-friendly dashboard. The core focus is on simplicity—transforming daily financial control from a tedious chore into a seamless habit.

Design Solution

The Outcome

Launch-Ready MVP

Designed a comprehensive product that addresses fundamental financial management needs, fully prepped for real-world user testing and development handoff.

Lower Barrier to Entry

Leveraging a clean UI and intuitive micro-patterns allows new users to navigate the core functionality instantly, eliminating the need for complex onboarding.

Scalability

The robust design system and information architecture facilitate seamless future updates, such as banking API integrations or shared household budgets, without breaking the existing layout.
